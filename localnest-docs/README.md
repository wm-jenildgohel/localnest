# LocalNest Docs

This is a Docusaurus site for LocalNest MCP.

## Development

```bash
cd localnest-docs
npm install
npm run start
```

## Production build

```bash
cd localnest-docs
npm run build
npm run serve
```

## Deployment

This site is configured for GitHub Pages style deployment:

```bash
cd localnest-docs
npm run deploy
```

Adjust `url` and `baseUrl` in `docusaurus.config.ts` if you deploy somewhere else.
